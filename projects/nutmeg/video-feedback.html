Guidelines for data collection

What characteristics does a useful video for training have?

The most important characteristic of a video is that atleast one object moves (either the camara or the ball, or something in the background like leggs or a dog). If everything is static, the video is simply a long image, and thus it is more useful to just make a picture (otherwise I have to examine the quality of each video). A video is better if there is more moving objects and there is as many other objects as possible. For example, multiple balls, multiple players, a moving camara and a moving ball, preferably were background objects appear and disappear during the video, introducing more variety.

+
  • Length: 2-10 seconds
  • Resolution: any
  • Frame rate: any
  • Key characteristic: At least one moving object
  • Camera movement: static or dynamic
  • Camera angle: any
  • Lighting conditions: any
  • Background: any
  • Soccer ball movement: dynamic or static
  • Soccer ball type: any

Examples of useful videos

Even though these videos are great as is, to make these videos even more useful the following could be done. For the first video, having more objects moving or moving the camera would be great. For the second video, having more objects in the background would be great as right now, the background is only the floor, which does not contain many distractions. For the third video, having more objects on the stairs would be great, as right now the background is only the stairs, which does not contain many distractions.
